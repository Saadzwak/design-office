# design://material-finishes

A working catalogue of interior-architecture finishes used in the Mood
Board surface. Each family lists 4-8 candidate products with their source,
typical application, sustainability notes, and a one-line "when to use"
cue for Opus.

These are real products from manufacturers; prices and lead times are
indicative orders of magnitude [TO VERIFY] unless marked otherwise.

## Floors

### Light wood-look LVT — for tech-startup / creative / healthcare

- **Amtico Signature — Worn Oak** (LVT, 7.5 mm)
  - Durable, easy install over existing slab
  - Class 33 commercial, warm tone, slip-resistant R10
  - Source : amtico.com
  - When to use : light wood feel where real timber is out of budget
- **Forbo Marmoleum — Natural Shell** (linoleum, 2.5 mm)
  - Fully biodegradable, 97 % natural raw materials
  - Low VOC, anti-bacterial
  - Source : forbo.com
  - When to use : healthcare, public sector, ESG-conscious clients
- **Bolon — Artisan Nature** (woven vinyl, 2.5 mm)
  - Heavy-duty, washable, 50 % recycled content
  - Source : bolon.com
  - When to use : high-traffic zones, creative agencies

### Solid wood — for law firm / bank / flagship reception

- **Dinesen Douglas plank** (solid Douglas fir, up to 15 m long)
  - Premium, natural oiled finish, ages beautifully
  - Source : dinesen.com
  - When to use : reception, executive corridors, library
- **Listone Giordano Atelier** (engineered oak, brushed)
  - Italian craftsmanship, multiple stains
  - Source : listonegiordano.com
  - When to use : law-firm corridors, client-facing floors

### Carpet tiles — for quiet + collab zones

- **Interface Composure** (500×500 mm tile, SKOA Grey, Deep Sky)

## Lighting

### Task lighting

- **Flos D'E-Light** — LED, adjustable, 2700-5000 K
- **Artemide Tolomeo** — Italian classic, 2700-4000 K LED
- **Baltus** — premium, solid brass, warm output
  - When to use : law-firm desks, boardrooms

### Pendant lighting

- **Muuto Ambit Rail** — tracks, customisable, Scandinavian
- **Vibia Wireflow** — sculptural, statement pieces
- **Nemo Lighting Cassina Linescapes** — linear, contemporary

### Feature lighting

- **Bocci 28 series** — blown glass pendants, cascades
  - When to use : reception, stair voids, signature zones
- **Foscarini Lumiere** — Murano glass icon
  - When to use : law-firm boardrooms, premium fit-outs

### Colour-temperature strategy

- 3000 K (warm, cosy) : reception, lounge, café, residential-style
  zones.
- 3500 K (neutral-warm) : meeting rooms, collab.
- 4000 K (neutral-cool) : open-plan work desks, task-focus areas.
- 5000 K+ (cool) : specialised production/workshop only.

CRI ≥ 90 in client-facing and creative zones ; ≥ 80 is acceptable in
back-office / circulations. Match EN 12464-1 lux targets.

## Sustainability markers to surface to the client

When relevant, agents should cite these labels in the mood board PDF :

- **Cradle-to-Cradle** — circular economy grade (Bronze/Silver/Gold/
  Platinum)
- **FSC** — Forest Stewardship Council (timber sourcing)
- **GreenGuard / GreenGuard Gold** — low-emission indoor products
- **DGNB, BREEAM, LEED, WELL** — building certifications (whole-fit-out
  level)
- **EPD (Environmental Product Declaration)** — LCA evidence
- **EPEA / Declare Red List** — material transparency
